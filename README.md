# Deeplang-plugin README

Deeplang is a new programming language for IoT device.
Offical Website: www.deeplang.org

## Features
- highlighter
- lsp (todo)
- repl (todo)
- debug (todo)

## Package
Install `yo` , `generator-code` and `vsce`
``` shell
npm install -g yo
npm install -g generator-code
npm install -g vsce
```
Package vsix
``` shell
vsce package
```
